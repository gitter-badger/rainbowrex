Rainbow Rex
=====================

An addictive, fast-paced :rainbow: colour picker game by @tongrhj
[Play in Browser](http://rainbowrex.herokuapp.com)

## To Install

```
npm install
```

## To build local server with watch HTML and CSS

```bash
$ ionic serve
```
